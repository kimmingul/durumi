## 4. Risks / unknowns

Couldn't resolve from code alone:

1. **Live mode + table widget**: invariant #11 says it works, zero Live coverage in `table-cell-edit.spec.ts`. Manual smoke needed.
2. **DOCX export with `durumi-asset://` images**: HTML keeps relative paths; didn't trace whether DOCX resolves the protocol. Could silently drop.
3. **Highlight live decoration vs `{== ==}` priority**: lezer order is correct (CriticMarkup `before:` Highlight, appendix B); decoration-plugin stacking when both fire on same range needs verification.
4. **Setext H2 underline `---` hiding** may collide with HR detection. Fixture test.
5. **`> [!NOTE]` + nested blockquote**: alert decoration semantics unclear.
6. **`applyCitations` with no `.bib`**: missing keys → `[?]` per doc; unverified whether References section silently drops or emits empty.
7. **`exportPreserveAnnotations` persistence**: no test asserts restart-survival.
8. **Korean IME matrix**: 6 marks × 3 modes = 18 cells. Can't enumerate breakage without running.

## How to sign

By submitting a pull request to the Durumi repository, You agree to the
above terms for any Contributions included in that pull request.

The simplest mechanism is to add a `Signed-off-by:` line to each commit
message (DCO-style), e.g.:

```
Signed-off-by: Your Name <your.email@example.com>
```

For pull requests on GitHub, you may also accept the CLA by commenting on
the pull request with the exact text:

```
I have read the CLA Document and I hereby sign the CLA
```

The repository maintainer will keep a record of CLA acceptances. In the
future, a CLA-bot (such as CLA Assistant) may be wired up to automate
this acknowledgement.

If You are contributing on behalf of an organization, please contact the
maintainer at the project email to arrange a separate Corporate CLA (CCLA).
